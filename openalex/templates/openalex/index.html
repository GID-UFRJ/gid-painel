PESQUISA da UFRJ

Os dados disponíveis neste painel foram coletados do OpenAlex, uma base de dados de produção científica que organiza e disponibiliza centenas de milhões de documentos científicos em diferentes formatos.

Para recuperar os dados da UFRJ, foi utilizado o critério endereço de filiação dos autores. Assim, os dados desta seção incluem documentos com pelo menos uma autoria filiada à UFRJ. Os dados foram obtidos programaticamente a partir da API da OpenAlex e foram modelados para amarzenamento em um banco de dados relacional. A equipe do GID atualiza esses dados a cada seis meses.

Para este painel, os dados estão apresentados em três blocos, Produção, Impacto e Colaboração. Para cada um deles selecionamos algumas variáveis que descrevem aspectos relevantes da pesquisa realizada na e pela UFRJ.

# Produção {% blocktrans asvar producao_nota %}

As áreas de conhecimento apresentadas nesta seção foram atribuídas pela OpenAlex, não sendo possível a comparação entre as áreas, pois os totais são impactados pelo número de periódicos que varia entre as áreas.

Já os dados de tipo de acesso do periódico foram reagrupados em duas categorias (restrito e aberto) e, assim, é possível observar a adesão da UFRJ ao movimento de acesso aberto. A autoria correspondente foi incluída nesta seção, pois ela revela a real liderança da UFRJ na pesquisa publicada.

# Impacto {% blocktrans asvar impacto_nota %}

Importante destacar que a citação não é sinônimo de qualidade da pesquisa publicada, mas do impacto ou da visibilidade dela. Há inúmeros fatores que contribuem para o aumento de citação de uma pesquisa publicada (Erikson & Erlandson, 2014).

Já o Índice H, originalmente de um/a determinado/a autor/a, pode ser utilizado para a análise de qualquer conjunto de documentos (Jones et al., 2011), leva em conta duas variáveis, a produção e a citação, e permite observar a produção-impacto sem a contribuição dos outliers. Este texto explica em detalhes as origens do índice H e como calculá-lo.
