Apesar do portal de dados abertos apresentar dados desde o ano 2004, este painel utiliza apenas os registros a partir de 2013, ano no qual a coleta passou a ser realizada através da plataforma Sucupira.

Os dados disponíveis neste painel foram coletados do Portal de Dados Abertos da Capes, que disponibiliza informações em série histórica sobre o corpo docente e discente de todos os programas de pós-graduação registrados no país (ver: https://dadosabertos.capes.gov.br/)

Para a recuperação dos dados dos programas foram baixados todos os arquivos csv relativos à avaliação da Pós-Graduação disponibilizados pela CAPES. Posteriormente, os registros relativos à UFRJ foram programaticamente extraídos de todos os arquivos através da linguagem Python. Os dados filtrados foram limpos e reestruturados, sendo posteriormente utilizados para a população do banco de dados utilizado para os painéis de pessoal e pós-graduação. À medida que a CAPES disponibilizar dados para anos mais recentes, a equipe do GID irá baixá-los, limpá-los e adicioná-los ao banco de dados para que as análises disponibilizadas estejam tão atualizadas quanto possível.

Para este painel, os dados estão apresentados em dois blocos, um com a visão geral do conjunto de programas e outro com informações individualizadas sobre os mesmos, para os quais foram selecionadas variáveis que descrevem aspectos relevantes sobre a Pós-Graduação na UFRJ.
